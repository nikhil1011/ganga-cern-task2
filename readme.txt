This NbExtension replaces all occurences of the string 'SWAN' in a rendered Markdown cell in a Jupyter notebook when it is enabled.

To install this extension,
 - download the zip of the code into a convenient location on your computer
 - extract it

To run this, execute the following commands
 - jupyter nbextension install /absolute/path/to/Task2	#example: jupyter extension install /home/nikhil/Documents/Task2
 - jupyter nbextension enable Task2/main
 
To disable,
 - jupyter nbextension disable Task2/main
